# 🎲 Sorteio de Números

Um gerador simples e elegante de números aleatórios únicos, desenvolvido como projeto de portfólio em HTML, CSS e JavaScript.

![Preview do Projeto](assets/screenshot.png) <!-- Substitua por uma screenshot real -->

## ✨ Funcionalidades

- **Sorteio automático**: Gera 6 números únicos entre 1 e 60.
- **Tema escuro/claro**: Alternância suave entre modos.
- **Interface responsiva**: Funciona perfeitamente em desktop e mobile.
- **Animações suaves**: Efeitos de entrada e hover interativos.
- **Ícones modernos**: Usando Font Awesome para melhor UX.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica.
- **CSS3**: Estilos modernos com variáveis CSS, flexbox e animações.
- **JavaScript ES6**: Lógica de sorteio e interatividade.
- **Font Awesome**: Ícones vetoriais.

## 🚀 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Ekho-c/sorteio-numeros.git
   ```

2. Abra o arquivo `index.html` no seu navegador.

3. Clique em "Sortear novamente" para gerar novos números!
